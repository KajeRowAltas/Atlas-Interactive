# Atlas Interactive

This repository is now organized as a project notebook and development ledger for Oji.

The repo no longer acts as a source-code monorepo. It tracks project state, priorities, current developments, open questions, and next actions across the main Oji-related initiatives.

## Structure

- `projects/` contains one markdown ledger per active project area.
- `projects/art-oji-beats.md` is the single retained art reference file from the previous repo structure.
- `logs/` is reserved for dated review notes and chronological updates.

## Project Scope

- `Atlas-Interactive` is the main Oji application project.
- `Oji-AI` is not a standalone top-level project in this repository.
- `Oji-AI` is tracked within `Atlas-Interactive` planning and development notes.
- The repository should stay documentation-first and should not be rebuilt into a code monorepo unless the project direction changes explicitly.

## Review Goal

This repo is prepared for review before commit. The intended next step is to confirm the notebook structure, then stage and commit the cleanup without pushing yet.
